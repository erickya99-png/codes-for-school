# Cybersecurity Projects

This repository contains two Python implementations for cybersecurity automation:

## Files

1. **MIDTERM EXAM -Port Scanner Implementation.py**: A Python-based port scanner that can scan specified ranges of ports on target hosts, identify open ports, and handle connections with proper error handling.

2. **test for CYB333 Security Automation2.py**: A TCP client-server implementation demonstrating basic network communication, with options to run as server, client, or demo mode.

## Usage

### Port Scanner
Run the port scanner script and follow the prompts to enter target host and port range.

### TCP Client-Server
Run with command line arguments:
- `python "test for CYB333 Security Automation2.py" server` - Run as server
- `python "test for CYB333 Security Automation2.py" client` - Run as client
- `python "test for CYB333 Security Automation2.py" demo` - Run both in demo mode

## Security Notice
These tools are for educational purposes only. Only scan authorized systems (localhost and scanme.nmap.org for port scanner).
